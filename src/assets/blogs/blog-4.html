Implementació de col·lisions i física en PacoMonkey

La física i les col·lisions són una part fonamental de PacoMonkey, ja que determinen com interactuen els objectes i el personatge principal. A Unity, fem servir el motor de física 2D per gestionar aquestes interaccions.

Configuració del sistema de col·lisions

Per gestionar les col·lisions, hem afegit components de Collider2D als objectes del món del joc. A més, utilitzem Rigidbody2D per aplicar forces i simular la gravetat al nostre personatge principal.

Exemple de codi per a les col·lisions


void OnCollisionEnter2D(Collision2D collision) {
    if (collision.gameObject.CompareTag("Enemy")) {
        // Restar vida o reiniciar el nivell
    }
}

Amb aquest codi, PacoMonkey pot detectar col·lisions amb enemics i altres objectes del joc, permetent-nos afegir efectes com perdre vida o reiniciar el nivell.
